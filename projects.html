Ready2Cook Mobile App

I developed a React Native app called Ready2Cook. This app helps users find recipes based on the ingredients they already have at home. By entering the ingredients you have, the app uses the Spoonacular API to search for recipes that make the most of those ingredients, minimizing food waste.

Dino Game

I recreated the popular Chrome dinosaur game with modern visuals using HTML, CSS, and JavaScript. JavaScript handles the creation of obstacles at set intervals, and I also built a separate script to control the dinosaur's jump when the spacebar is pressed. When the dino collides with an obstacle, the game enters a "game over" state, and a new background is displayed. Additionally, I used CSS for the dino's animations, employing keyframes and sprites to animate its running, jumping, and dying actions.
